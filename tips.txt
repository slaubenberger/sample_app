Stubs/Guard
-----------

chmod +x $rvm_path/hooks/after_cd_bundler

cd <yourApp>
bundle install --without production --binstubs=./bundler_stubs

bundle install --binstubs=./bundler_stubs
bundle --binstubs

-> Add Guard (look Gemfile)

bundle install

bundle exec guard init rspec
